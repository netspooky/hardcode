ิอออออออออออออออออออออออออออออออออออออออออออออออออออออออออออออออออออออออออออออพ

Moderator is just a little  64k intro first shown at the euskal party V,  where
it got the 2nd place.

About 2 weeks of hard coding, but finally I didn't need to code any part of the
intro at the party place.  Some features: perspective corruption,  Z-Fuckering,
fuck  shading,   camera  b-spline  interputation   (using  heuristic  recursive
algorithms, too) and mpeg layer 69 compression algorithm.

There is a hidden key on the intro, if you find it you could access to inedited
pictures of  Pamela Anderson fucking  with different animals.  There is also an
.AVI movie from the last Long Dan Silver film!!!!

MOV  and  MDEC  versions  from the  film  are  available too,  at the following
address:

        http://bebe.uv.es/~fgonzale/tlotb.html
