How to play:
	Select rotation	- LeftArrow, RightArrow
	Select speed	- UpArrow, DownArrow
	Move left	- LeftArrow
	Move right	- RightArrow
	Rotate		- UpArrow
	Move down	- DownArrow
	Exit		- esc

Note: run in fullscreen

P.S. There's a bug, speed doesn't change sometimes.
